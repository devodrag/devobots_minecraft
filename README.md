# DevoBots v1.5 [Terminal + Telegram Sync]

Продвинутый инструмент для одновременного управления группой Minecraft-ботов на базе Node.js и Mineflayer. Позволяет координировать действия аккаунтов как через локальную консоль, так и удаленно с помощью Telegram-бота.

---

## ⚡ Основной функционал

* **Двойной контроль:** Полная синхронизация между командами в терминале и интерфейсом Telegram.
* **Умная навигация:** Интегрированный плагин `mineflayer-pathfinder` с поддержкой команд Baritone (`#baritone goto X Y Z` / `#baritone stop`).
* **Модуль DevoClient:** Настраиваемый алгоритм спама:
* Массовый спам (`all`) — каждый активный бот отправляет сообщение заданное количество раз.
* Точечный спам (по номеру) — флуд запускается строго с одного выбранного аккаунта.


* **Динамический конфиг:** Изменение версии Minecraft и задержки подключения (Connect Delay) прямо во время работы программы.
* **Приватность:** Встроенная проверка `adminId` в Telegram — бот будет реагировать только на сообщения владельца.

## 🛠 Установка и запуск

1. Убедитесь, что на компьютере установлен **Node.js**.
2. Склонируйте репозиторий или скачайте файлы проекта.
3. Установите необходимые зависимости, выполнив команду в терминале:

```bash
npm install mineflayer mineflayer-pathfinder node-telegram-bot-api

```

4. Откройте файл `devobots.js` и в блоке `CONFIG` укажите ваш токен от `@BotFather` и ваш Telegram ID (можно узнать через `@userinfobot`):

```javascript
const CONFIG = {
    version: '1.8.9',
    connectDelay: 0.5,
    telegramToken: 'ВАШ_ТОКЕН_БОТА',
    adminId: ВАШ_TELEGRAM_ID
};

```

5. Запустите скрипт:

```bash
node devobots.js

```

## 📱 Управление через Telegram

После отправки команды `/start` или `/menu` бот пришлет интерактивный пульт. Доступны следующие текстовые команды:

* `/add [Ник]` — добавить новый ник в базу `accounts.json`.
* `/del [Номер]` — удалить ник из базы по его порядковому номеру.
* `/version [Версия]` — изменить версию игры (например, `1.12.2`, `1.16.5`).
* `/delay [Секунды]` — изменить интервал между заходами ботов.
* `/startall [IP:Порт]` — запустить процесс подключения всех ботов на сервер.
* `/msg [Текст]` — отправить сообщение или команду (включая `#baritone` и `!devo`) со **всех** ботов.
* `/msgone [Номер] [Текст]` — отправить сообщение или команду только с **одного** конкретного бота.
